# Security Policy

## Reporting a Vulnerability

Please report suspected security issues through GitHub Issues:

- https://github.com/NathanaelS/Kanoli/issues

When possible, include:

- A clear description of the issue
- Reproduction steps
- Potential impact
- Affected files or components

## Scope

This policy applies to the main Kanoli repository and released source code.

## Response

Security reports are reviewed as time permits. Valid reports will be triaged and prioritized based on impact.
