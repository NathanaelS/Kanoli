# Kanoli Flutter Port

This project is the Flutter recreation of Kanoli and is intentionally isolated from the original Swift project.

## Phase 2 status

- Cross-platform Flutter scaffold created (macOS, Windows, Linux, iOS, Android).
- Core architecture folders established (`core`, `domain`, `data`, `features`).
- Baseline app shell, theming, routing hook, logging, and uncaught error handling added.
- Widget smoke test updated for the Phase 2 app shell.
